# Results Directory

This folder contains all outputs generated by the CKD prediction notebooks.

## CSV Files (5 total)

Essential data files for analysis and reporting:

### From Notebook 1: EDA and Preprocessing
- `dataset_summary_table.csv` - Dataset statistics and summary metrics
- `missing_data_summary.csv` - Missing data patterns by feature

### From Notebook 2: Model Training and Evaluation
- `model_comp_results.csv` - Raw model comparison metrics (all models)
- `detailed_model_metrics.csv` - Formatted metrics with clinical thresholds

### From Notebook 3: SHAP Analysis and Interpretation
- `hyperparameter_tuning_summary.csv` - Best hyperparameters and improvements

## Reproducibility

All results can be regenerated by running the notebooks in sequence:
1. `01_EDA_and_Preprocessing.ipynb` → 7 plots + 2 CSVs
2. `02_Model_Training_and_Evaluation.ipynb` → 2 plots + 2 CSVs
3. `03_SHAP_Analysis_and_Interpretation.ipynb` → 9 plots + 1 CSV + 3 PKL models

Results are deterministic due to fixed random seeds (`random_state=42`).
